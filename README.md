# Evaluation of Self-Supervised Vision Transformers for Image Classification in Automated Visual Inspection of Electric Motor Components

### Research Objective: 

The objective of this study is to assess the effectiveness of two models for visual inspection tasks within manufacturing industries: the vision transformer-base (ViT-B) and a self-supervised variant called the self-distillation with no labels version 2 (DINOv2) model. The
motor brush holder (MBH) dataset provided by Wang et al. is used in this work. Furthermore, the study provides an overview of the self-supervised vision transformers and their potential compared to residual network 50 (ResNet50).

The primary research goal is to train and evaluate these DL models on the MBH dataset, with the established ResNet50 model as the benchmark. This comprehensive approach not only aims to advance predictive visual inspection methods but also emphasizes the significance of selfsupervised learning in optimizing model performance to ensure manufacturing quality.
